# Wean Project Art Theme

## Theme Name
**"Friendly Reduction"** - Absurdist wellness mascot aesthetic

## Visual Language
- Friendly, slightly unsettling creatures that feel supportive
- Leaf/plant motifs representing natural substances
- Measuring/reduction visual metaphors (cups with markings, gradients)
- Vibrant greens and calming blues
- Soft gradients, non-clinical feel
- "Corporate wellness designed by a 5-year-old CEO" energy

## Tone
- Encouraging but not patronizing
- Quirky but trustworthy
- Harm reduction philosophy (non-judgmental)
- Celebrates small wins, not perfection

## Usage Guidelines

### Ticket Categories
| Category | Art Direction |
|----------|---------------|
| Core Features | Green leaf mascot, measuring imagery |
| Premium Features | Gold/yellow accents, achievement badges |
| Bugs | Red-tinted mascot looking concerned |
| Chores | Gray-toned, calm maintenance vibes |

### Color Palette
- Primary Green: `#22c55e` (growth, wellness)
- Calming Blue: `#3b82f6` (trust, stability)
- Warm Yellow: `#fbbf24` (premium, achievement)
- Soft Red: `#f87171` (alerts, not alarming)
- Neutral Gray: `#64748b` (infrastructure)
